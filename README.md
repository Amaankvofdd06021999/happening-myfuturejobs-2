# MYFutureJobs - National Employment Portal

A modern, AI-powered employment portal for Malaysia, built with React, TypeScript, and TanStack Start.

## 🚀 Features

### Landing Page
- **Hero Section** with AI-powered job search
- **Top Employers Showcase** - Featured companies actively hiring
- **Events & Programs** - Smart event recommendations based on user profile
- **AI Features Section** - Showcasing intelligent matching capabilities
- **Industry Solutions Marquee** - Tailored solutions for every Malaysian industry
- **Success Stories** - Real testimonials with video content
- **Career Services Grid** - Complete toolkit for career development
- **ROI Metrics** - Demonstrable improvements for employers
- **Mobile App Section** - Promoting mobile-first experience
- **Resources Hub** - Guides, webinars, and templates

### AI-Powered Features
- **Context-Aware Chatbot** - Provides section-specific assistance
- **Smart Job Matching** - AI analyzes 400M+ profiles
- **Career Signal Score** - Real-time market positioning
- **CV Optimization** - AI-powered resume improvements
- **Bias Detection** - Ensures fair and inclusive hiring
- **Predictive Matching** - ML models predict candidate success

### Dashboard Features
- **Job Seeker Dashboard** - Profile, applications, career score
- **Employer Dashboard** - Vacancy management, AI screening tools
- **Officer Dashboard** - Case management and analytics

## 🛠️ Tech Stack

- **Framework**: TanStack Start (React + Vite)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Custom components + shadcn/ui
- **State Management**: Zustand
- **Routing**: TanStack Router
- **Icons**: Lucide React
- **Animations**: CSS animations with marquee effects

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/Amaankvofdd06021999/happening-myfuturejobs-2.git
cd happening-myfuturejobs-2
```

2. Install dependencies:
```bash
npm install
# or
bun install
```

3. Run the development server:
```bash
npm run dev
# or
bun dev
```

The application will be available at `http://localhost:8080`

## 🚢 Deployment

### Vercel Deployment

1. Connect your GitHub repository to Vercel
2. Configure build settings:
   - Build Command: `npm run build`
   - Output Directory: `dist`
3. Deploy!

### Environment Variables

No environment variables are required for the basic setup. The application uses mock data for demonstration purposes.
